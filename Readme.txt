# GoConfig
CS2 自动配置文件

## 如何使用

1. 下载并解压，运行 `GoDeploy.bat`
2. 重新打开游戏，并打开控制台
3. 当 "Goconfig is ready" 出现在控制台时，说明配置已生效


## 推荐配置

1. 使用 autoexec.cfg
2. 游戏使用无框窗口模式，提高桌面和游戏间切换速度（不会黑屏）
3. eDPI 1200 = 800(鼠标) x 1.5(sensitivity)
